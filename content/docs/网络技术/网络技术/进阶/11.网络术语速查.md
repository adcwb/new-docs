## 通用概念

- **泛洪（Flooding）**：交换机和网桥使用的数据传递技术，将某接口收到的数据从除该接口外的所有接口发出。
- **冗余（Redundancy）**：出于安全和可靠性考虑，人为对关键部件或功能重复配置；故障时备援部件及时接管，减少故障时间。
- **协议栈**：网络中各层协议的总和，形象反映文件传输过程（上层到底层、再从底层到上层），应用最广的是因特网协议栈。
- **广播域（Broadcast Domain）**：某设备同时向网络中所有其他设备发送数据，数据能广播到的范围。
- **冲突域（Collision Domain）**：CSMA/CD 网络中两台计算机同时通信会发生冲突的范围；以集线器连接的网段仍是一个冲突域。
- **与运算**：对应位全为 1 结果为 1，否则为 0（用于 IP 与掩码求网络地址）。
- **无盘系统**：由无盘工作站组成的局域网，工作站无需任何外存即可启动并运行操作系统，依赖专门软件系统支撑。
- **5-4-3 规则 / 最小帧**：以太网早期共享介质组网的经典约束概念。

## 路由相关术语

- **邻居（neighbor）**：相互发送过 Hello 的相邻路由器。
- **邻接（adjacency）**：可以发送 DD、LSA 信息的关系（真正交换路由）。
- **链路（link）**：设备之间的物理连接（OSPF 中等同于接口）。
- **LSA**：链路状态信息（网段信息、路由条目）。
- **LSDB**：LSA 的数据库（链路状态数据库）。
- **AS（自治系统）**：隶属于同一管理机构的路由器集合，运行同一协议即在一个自治系统内。
- **ABR**：区域边界路由器；**ASBR**：自治系统边界路由器。
- **VLSM**：可变长子网掩码；**CIDR**：无类域间路由；**路由聚合**：聚合到主类网络。
- **Router ID**：32 位二进制、用 IP 写法表示的标识，通常用 loopback 0 回环口的 IP 作为 Router ID。
- **缺省路由（Default Route）**：主机/路由器无特定路由时的转发规则，给出默认下一跳；路由表中无匹配项的数据包都走缺省路由（`0.0.0.0/0`）。
- **IGP**：内部网关协议（RIP、IS-IS、OSPF）；**EGP**：外部网关协议（BGP）。IGP 负责计算，EGP 负责选路。
- **域间路由**：自治系统之间通过域间路由协议交换路由，Internet 上的事实标准是 BGP-4。
- **虚链路（Virtual-link）**：在两台 ABR 之间穿过一个非骨干（转换）区域建立的逻辑点对点连接，被视为属于骨干区域，用于解决区域无法与骨干直连的问题。

## ACL 与策略相关

- **基本 ACL**：只限定源地址。
- **高级 ACL**：限定两个以上条件（源/目的 IP、端口、协议号等）。
- **比较关键字**：`eq`（等于）、`gt`（大于）、`lt`（小于）。
- **网络位**又称**前缀**；相关工具有前缀列表、路由策略。

## IPv6 简述

IPv4 地址即将枯竭，根本解决方案是 **IPv6**。IPv6 地址有 **128 个二进制位**，约 $2^{128}$ 个地址，可彻底解决地址紧张问题。运行 IPv6 的路由器同样维护邻居表、路由表、链路状态数据库表（在链路状态协议下）。
