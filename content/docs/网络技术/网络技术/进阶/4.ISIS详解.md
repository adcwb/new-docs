IS-IS 与 OSPF 一样，是基于链路状态、使用最短路径优先（SPF）算法的 IGP。这篇讲 IS-IS 的特点、NSAP/NET 地址结构、与 OSPF 的差异以及术语对照。

## IS-IS 概述

**IS-IS（Intermediate System to Intermediate System）**最初由 ISO 为其无连接网络协议 CLNP 设计。IETF 在 RFC 1195 中对它扩充修改，使其能同时应用于 TCP/IP 和 OSI 环境，修订后称为**集成化 IS-IS**。由于简便、扩展性强，目前在大型 ISP 网络中被广泛部署。

基本属性：

- **无类、链路状态、内部网关协议（IGP）**。
- 默认优先级：**15**。
- 工作原理同样是：建邻居、生拓扑、算路由。
- 工作在**二层**（OSPF 工作在 IP 层）。
- 三张表：路由表、邻居表、数据库表（LSDB）。
- 三种邻居状态：`Down`、`Init`、`Up`。

常用命令：

```text
import-route <protocol>        # 引入其他路由
default-route-advertise        # 引入默认路由
display isis error             # 查看报错信息
summary 10.0.0.0 255.0.0.0     # 路由聚合
```

## NSAP 与 NET 地址

IS-IS 使用 **NSAP（Network Service Access Point，网络服务接入点）**地址，结构如下：

- **IDP**：相当于 IP 的主网络号，由 ISO 规定，含 AFI（地址分配机构和格式）和 IDI（标识域）。
- **DSP**：相当于 IP 的子网号和主机地址，由 High Order DSP（分割区域）、System ID（区分主机）、SEL（指示服务类型）组成。
- **Area Address（区域地址）**：由 IDP 和 DSP 中的 High Order DSP 组成，相当于 OSPF 的区域编号。
- **System ID**：区域内唯一标识主机或路由器，固定 48bit（6 字节）。
- **SEL**：类似 IP 的"协议标识符"，在 IP 上 SEL 均为 `00`。

**NET（Network Entity Title，网络实体名称）**指设备本身的网络层信息，可看作 SEL=00 的特殊 NSAP。长度 8~20 字节。在路由器上配置 IS-IS 时只需考虑 NET。

NET 格式（IP 环境下）：**区域 ID（1 字节）+ 系统 ID（6 字节）+ SEL（1 字节）**，每位都是一个十六进制数字，SEL 总为 `00`。

```text
49.0001.0000.0000.0001.00
└Area ID┘└── System ID ──┘└N-SEL┘
```

> 一个十六进制数代表四个二进制位；`47`/`49` 等代表私网地址。配置时 NET 最多配 3 个，且多个 NET 的 System ID 必须相同。

其他概念：**TLV**（Type-Length-Value，类型-长度-值）、**PDU**（协议数据单元）、支持无连接网络协议（CLNP）、使用 SPF 算法。

## 与 OSPF 的差异

| 对比项 | OSPF | IS-IS |
| --- | --- | --- |
| 工作层 | IP 层 | 二层 |
| 支持网络 | 只支持 IP | 支持 IP，也支持 CLNP |
| 伪节点选举 | 选 DR 和 BDR | 只选 DIS，不选备份 |
| 邻接关系 | DRother 间只是邻居 | 每台路由器之间都是邻接关系 |
| DIS/DR 抢占 | DR 不抢占 | DIS 会抢占 |
| 优先级 0 | 不参选 | 参选 |
| 默认优先级 | 10 | 15 |
| 路由表更新 | 30 分钟 | — |

术语对照：

| OSPF | IS-IS |
| --- | --- |
| Router（路由器） | IS（中间系统） |
| Host（主机） | ES（终端系统） |
| DR | DIS |
| RID | System ID |
| AS（自治系统） | Routing Domain（路由域） |
| MAC | SNPA（子网连接点） |
| packet（包） | NPDU |
| frame（帧） | SNPDU |
| 骨干区域 Area 0 | Level 2 subdomain |
| 非骨干区域 | Level 1 area |
| ABR | Level 1-2 路由器 |
| LSA | LSP |
| Hello | Hello |
| DD | CSNP |
| LSR | PSNP |
| LSU | LSP |
| LSAck | 无（确认都用 LSP/PSNP） |

> 与 OSPF 不同，IS-IS 中任意路由器之间都是邻接关系，且每个区域都有自己独立的 LSDB。
