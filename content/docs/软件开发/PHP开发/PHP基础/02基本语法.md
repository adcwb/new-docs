这篇讲 PHP 代码的基本结构、注释的写法、大小写敏感规则，以及两种输出语句 `echo` 和 `print` 的区别。读完你能写出结构正确的 PHP 脚本。

## PHP 代码标记

PHP 脚本以 `<?php` 开头，以 `?>` 结尾。PHP 代码可以放在文档的任何位置，通常与 HTML 混写：

```php
<?php
// 这里是 PHP 代码
?>
```

{{< callout type="warning" >}}
PHP 语句必须以分号 `;` 结尾。如果代码块末尾的 `?>` 是文件的最后一行，它自带分号效果——但养成每条语句都加分号的习惯，可以避免很多意外错误。
{{< /callout >}}

## 注释

注释不会被程序执行，唯一的作用是让代码更容易理解。PHP 支持三种注释写法：

单行注释：

```php
<?php
// 这是单行注释（推荐）
#  这也是单行注释（Unix Shell 风格）
?>
```

多行注释：

```php
<?php
/*
 * 这是多行注释块
 * 适合写函数说明或大段解释
 */
?>
```

## 大小写敏感规则

PHP 的大小写规则需要分开记：

**关键词、函数名、类名**——不区分大小写：

```php
<?php
echo "Hello PHP!<br>";
ECHO "Hello PHP!<br>";   // 同样有效
Echo "Hello PHP!<br>";   // 同样有效
?>
```

**变量名**——严格区分大小写：

```php
<?php
$color = "red";
echo $color;   // 输出 red
echo $COLOR;   // 未定义，不输出任何内容（三个是不同的变量）
echo $coLOR;   // 未定义
?>
```

> 建议统一大小写风格：关键词小写（`echo`、`if`、`function`），变量用小驼峰（`$userName`），这是 PHP 社区的主流习惯。

## echo 与 print

PHP 有两种基本输出语句，都是语言结构而非函数：

| 特性 | `echo` | `print` |
| --- | --- | --- |
| 输出数量 | 可输出多个字符串（逗号分隔） | 只能输出一个字符串 |
| 返回值 | 无返回值 | 始终返回 `1` |
| 速度 | 略快 | 略慢 |
| 括号 | 可选：`echo` 或 `echo()` | 可选：`print` 或 `print()` |

echo 示例：

```php
<?php
echo "Hello world!<br>";
echo "这段话", "由", "多个", "字符串", "串接而成。";
?>
```

print 示例：

```php
<?php
print "Hello world!<br>";
print "I'm learning PHP.";
?>
```

{{< callout type="tip" >}}
大多数场景下 `echo` 是首选——它更快且支持多参数。`print` 返回 1 的特性极少用到，只在需要将输出嵌入表达式时考虑。
{{< /callout >}}

## 一句话小结

PHP 代码写在 `<?php ... ?>` 标记内，每条语句以分号结尾。变量名分大小写，关键词不分。输出用 `echo`（多数场景首选）或 `print`。下一篇讲 [变量与常量](../03变量与常量/)。
