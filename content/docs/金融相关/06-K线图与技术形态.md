K 线（Candlestick）起源于 18 世纪日本大米期货市场，是全球最通用的价格可视化工具。本文从单根 K 线结构出发，逐步覆盖经典形态、量价关系和趋势判断方法。

## 单根 K 线结构

```text
        │  ← 上影线（最高价 → 收盘价 或 开盘价）
       ┌┴┐
       │  │  ← 实体（开盘价 ↔ 收盘价）
       │  │
       └┬┘
        │  ← 下影线（收盘价 或 开盘价 → 最低价）

阳线（红/空心）：收盘 > 开盘（上涨）
阴线（绿/实心）：收盘 < 开盘（下跌）
```

| 组成 | 含义 |
| :--- | :--- |
| 实体长度 | 买卖力量对比强度，越长越有决断力 |
| 上影线长度 | 上方遭遇的抛压，越长说明上攻被压制 |
| 下影线长度 | 下方获得的支撑，越长说明买盘承接较好 |

---

## 单根 K 线经典形态

### 看涨形态

**大阳线（Long White Candle）**

```text
实体占整根 K 线 70% 以上，无或极短影线。
含义：买方完全主导，趋势强劲；突破关键压力位时信号更强。
```

**锤头线（Hammer）**

```text
       ┌┤  ← 很短的实体（阳或阴均可）
       └┤
        │
        │  ← 下影线长度 ≥ 实体 2 倍
        │

出现条件：下跌趋势末端
含义：低位遭到大量买盘承接，多方开始反击，潜在见底信号。
```

**倒锤线（Inverted Hammer）**

```text
        │
        │  ← 上影线长度 ≥ 实体 2 倍
       ┌┤
       └┤  ← 很短的实体

出现条件：下跌趋势末端（次日需用阳线确认）
含义：试探上方压力但获得支撑，潜在反转。
```

**启明星（Morning Star）** — 三根组合

```text
  ┌──┐         ← 大阴线（下跌动能强）
  └──┘
       ┌┤      ← 小实体（多空犹豫，跳空低开或平开）
       └┤
          ┌──┐ ← 大阳线（收盘回升至阴线实体中部以上）
          └──┘
出现条件：下跌趋势末端
含义：下跌动能耗尽，多方开始接管，是较强的见底信号。
```

**红三兵（Three White Soldiers）**

```text
  ┌┐
  └┤┌┐
    └┤┌┐
      └┘
三根连续阳线，每根开盘在前根实体内，收盘逐步抬高。
含义：多方连续发力，趋势扭转信号强。
```

### 看跌形态

**大阴线（Long Black Candle）**

与大阳线对称，空方完全主导，跌势强劲。

**射击之星（Shooting Star）**

```text
        │
        │  ← 上影线 ≥ 实体 2 倍
       ┌┤
       └┤  ← 很短的实体（阴线更典型）

出现条件：上涨趋势末端（锤头线的镜像）
含义：高位冲高被大量抛售压回，潜在见顶信号。
```

**上吊线（Hanging Man）**

```text
       ┌┤  ← 很短的实体
       └┤
        │
        │  ← 下影线长

形态与锤头线相同，区别在于出现在上涨趋势末端。
含义：高位出现长下影线，意味着有人趁反弹出货。
```

**黄昏星（Evening Star）** — 三根组合，启明星的镜像

```text
   ┌──┐         ← 大阳线（上涨动能强）
   └──┘
         ┌┤     ← 小实体（多空犹豫，跳空高开）
         └┤
              ┌──┐ ← 大阴线（收盘跌入阳线实体中部以下）
              └──┘
含义：上涨动能耗尽，空方接管，是较强的见顶信号。
```

**乌云盖顶（Dark Cloud Cover）** — 两根组合

```text
   ┌──┐
   └──┤┌──┐ ← 阴线开盘在阳线上方，收盘深入阳线实体 50% 以上
      └──┘
含义：上涨遭遇大量抛售，趋势可能反转。
```

**三只乌鸦（Three Black Crows）**

红三兵的镜像，三根连续阴线，空头持续施压。

### 方向不明形态

**十字星（Doji）**

```text
        │
       ─┼─  ← 开盘 ≈ 收盘（几乎无实体）
        │
```

开盘价与收盘价极接近，形成十字形。多空力量均衡，市场方向迷茫，常出现在趋势末端，是重要的**转折预警信号**（需结合前后 K 线确认）。

- **墓碑十字**：长上影 + 无下影，上方压力巨大
- **蜻蜓十字**：无上影 + 长下影，下方支撑较强
- **长脚十字**：上下影均长，多空均不占优

## 量价关系分析

量价是技术分析的双引擎，价格告诉你走了哪里，成交量告诉你走得有多"真实"。

### 四种量价组合

| 量 | 价 | 含义 | 操作参考 |
| :--- | :--- | :--- | :--- |
| 放量 | 上涨 | 多方主动，趋势强 | 顺势持有/追入 |
| 缩量 | 上涨 | 上方压力减小，趋势弱 | 谨慎，可能滞涨 |
| 放量 | 下跌 | 空方主动出货，下跌趋势强 | 减仓/回避 |
| 缩量 | 下跌 | 抛压减弱，跌势末端可能 | 观望，等量能配合止跌 |

### 量价背离

```text
顶背离（看跌）：价格创新高，但成交量未创新高（甚至萎缩）
  → 上涨失去量能支撑，主力借高位减仓

底背离（看涨）：价格创新低，但成交量未创新低（甚至萎缩）
  → 下跌抛压枯竭，换手趋于完成
```

### 天量见天价 / 地量见地价

```text
天量：某日成交量极度放大（是近期均量 3 倍以上）
  → 换手充分，高度换手后价格往往短期见顶

地量：成交量极度萎缩（是近期均量 1/3 以下）
  → 市场参与者撤离，底部蓄势，往往预示大行情来临
```

## 均线系统

### 移动平均线（MA）

```text
MA(N) = 前 N 个交易日收盘价之和 ÷ N

常用周期及含义：
  MA5   → 一周线（超短线参考）
  MA10  → 两周线（短线趋势）
  MA20  → 月线（主要支撑/压力线）
  MA60  → 季线（中期趋势）
  MA120 → 半年线
  MA250 → 年线（牛熊分水岭）
```

### 均线信号

**金叉（看涨）**：短期均线从下向上穿越长期均线

```text
MA5 上穿 MA20（又称"黄金交叉"）→ 短线做多信号
```

**死叉（看跌）**：短期均线从上向下穿越长期均线

```text
MA5 下穿 MA20（"死亡交叉"）→ 短线离场信号
```

**均线多头排列**：短周期均线在上，长周期在下（MA5 > MA10 > MA20 > MA60）

```text
→ 多头市场格局，各均线形成支撑，逢回调买入机会
```

**均线空头排列**：短周期均线在下，长周期在上

```text
→ 空头市场格局，各均线形成压力，逢反弹减仓机会
```

### 指数移动平均线（EMA）

EMA 给近期价格更大权重，比 MA 更灵敏：

```text
EMA(N) = 前一日 EMA × (N-1)/(N+1) + 当日收盘价 × 2/(N+1)
```

MACD、布林带等指标的计算都基于 EMA。

---

## MACD 详解

MACD（Moving Average Convergence/Divergence，指数平滑移动平均线）是趋势确认类指标，适合判断中级趋势，不适合震荡市。

```text
EMA12 = 12 日指数移动平均
EMA26 = 26 日指数移动平均
DIF   = EMA12 − EMA26（快线）
DEA   = DIF 的 9 日 EMA（慢线/信号线）
MACD 柱 = 2 × (DIF − DEA)（红绿柱，能量棒）
```

### 主要信号

| 信号 | 描述 | 强度 |
| :--- | :--- | :---: |
| 金叉 | DIF 上穿 DEA | ★★★ |
| 死叉 | DIF 下穿 DEA | ★★★ |
| 零轴以上金叉 | DIF > 0 时金叉 | ★★★★ |
| 顶背离 | 价格新高，MACD 不新高 | ★★★★★ |
| 底背离 | 价格新低，MACD 不新低 | ★★★★★ |

### MACD 背离示例

```text
顶背离（做空信号）：
  第一波高点：股价 50，DIF = 2.5
  第二波高点：股价 55（更高），DIF = 1.8（更低）← 背离！
  → 价格在上涨，但多头动能在减弱，做空或离场

底背离（做多信号）：
  第一波低点：股价 30，DIF = -2.0
  第二波低点：股价 25（更低），DIF = -1.2（更高）← 背离！
  → 价格在下跌，但空头动能在减弱，做多或建仓
```

## 布林带（BOLL）

布林带用统计学标准差衡量价格波动范围：

```text
中轨 = MA20（20 日均线）
上轨 = MA20 + 2σ（2 倍标准差）
下轨 = MA20 − 2σ

σ = √[Σ(收盘价 - MA20)² / 20]
```

### 布林带交易信号

| 形态 | 含义 |
| :--- | :--- |
| 带宽收窄（"口" 字形） | 低波动蓄势，即将爆发，方向未定 |
| 价格上穿上轨 | 强势突破，趋势上行确认；若迅速回落则为假突破 |
| 价格下穿下轨 | 弱势破位；超卖后可能反弹 |
| 价格在上轨附近游走 | 强势多头行情，不急于做空 |
| 价格在下轨附近游走 | 弱势空头行情，不急于做多 |

## 支撑位与压力位

### 什么是支撑 / 压力

```text
支撑位（Support）：价格下跌时容易停止并反弹的区域
  → 大量买盘聚集，历史上多次有效的低点

压力位（Resistance）：价格上涨时容易停止并回落的区域
  → 大量卖盘聚集，历史上多次有效的高点

支撑与压力会相互转换：
  被有效突破的压力位 → 变为新的支撑位（"踩实"）
  被有效跌破的支撑位 → 变为新的压力位
```

### 常见支撑 / 压力来源

1. **历史整数关口**：如 60 元、100 元（心理价位）
2. **前高 / 前低**：历史成交密集区
3. **均线**：MA20、MA60 常作为动态支撑/压力
4. **缺口（Gap）**：价格跳空留下的空白区域，往往会被"回补"
5. **黄金分割位（斐波那契）**：
   ```text
   常用回调比率：
     23.6%、38.2%、50.0%、61.8%（黄金比例）、78.6%
   
   计算：上涨从 A 到 B，回调目标位 = B − (B−A) × 比率
   ```

## 趋势线与通道

```text
上升趋势线：连接上涨过程中的至少两个有效低点
  → 价格有效回踩不破为做多机会；跌破趋势线是出场信号

下降趋势线：连接下跌过程中的至少两个有效高点
  → 价格有效反弹不破为做空机会；突破趋势线是反转信号

等距通道（Channel）：将趋势线平行移动到对侧
  → 在通道内波段操作，下轨附近买、上轨附近卖
```

## K 线分析注意事项

{{< callout type="warning" >}}
K 线形态是概率工具，不是必然信号。以下几点是实战中最容易忽略的陷阱：

1. **周期越大，信号越可靠**：日线 K 线的信号强于 1 分钟 K 线。
2. **必须结合成交量**：没有量能配合的形态信号可信度大打折扣。
3. **需要看形态出现的位置**：锤头线出现在高位叫上吊线，意义完全相反。
4. **右侧确认原则**：等形态完成后再入场，不要预判尚未出现的形态。
5. **止损优先**：所有形态都有失效概率，做好止损才是生存之本。
{{< /callout >}}
