本文以 A 股上市公司中国稀土（SZ000831）2026-06-23 的盘口数据为例，逐一拆解每项参数的含义与计算公式。当日该股以 65.00 元高开，最终跌停收于 59.67 元（-10.00%），在 2026-06-22 创历史新高 66.88 元后次日即封死跌停，是一个典型的"高位反转"案例。

## 价格类参数

### 今开 / 开盘价

集合竞价阶段（9:15–9:25）形成的第一个成交价。

```text
当日开盘价 = 集合竞价撮合成交价
中国稀土：今开 = 65.00 元
```

开盘价与昨收价之差称为**跳空**：

- 今开 > 昨收 → 跳空高开（高开 65.00 vs 昨收 66.30，实际低开 -1.30）
- 今开 < 昨收 → 跳空低开

### 昨收 / 前收盘价

```text
昨收 = 上一交易日收盘价 = 66.30 元
```

绝大多数当日指标的**基准参考价**均以昨收计算。

### 涨停价 / 跌停价

A 股主板涨跌幅限制为 ±10%（科创板、创业板为 ±20%，ST 股为 ±5%）：

```text
涨停价 = 昨收 × (1 + 涨跌幅限制) = 66.30 × 1.10 = 72.93 元
跌停价 = 昨收 × (1 - 涨跌幅限制) = 66.30 × 0.90 = 59.67 元
```

当收盘价 = 跌停价时，称为**封跌停**。本例收盘 59.67 = 跌停价，属于全天跌停。

### 涨跌幅

```text
涨跌幅 = (当前价 - 昨收) / 昨收 × 100%
       = (59.67 - 66.30) / 66.30 × 100% ≈ -10.00%
```

### 振幅

衡量当日价格波动范围：

```text
振幅 = (最高价 - 最低价) / 昨收 × 100%
     = (65.00 - 59.67) / 66.30 × 100% ≈ 8.04%
```

振幅越大，说明当日价格波动越剧烈。

### 均价

成交额加权平均价：

```text
均价 = 当日成交额 / 当日成交量
     = 48.49 亿 / (78.49 万手 × 100 股/手) ≈ 61.78 元
```

均价可作为判断当日多空成本的参考线：股价在均价以下运行说明持仓者整体亏损。

### 52 周最高 / 最低

过去 52 个自然周（约一年）内的价格极值：

```text
52 周最高 = 66.88 元（2026-06-22 创历史新高）
52 周最低 = 34.01 元
```

股价突破 52 周新高往往是强势信号；从 52 周高位大幅回落则需警惕趋势反转。

## 成交量 / 资金类参数

### 成交量与成交额

```text
成交量 = 78.49 万手（每手 100 股）
成交额 = 成交量 × 均价 ≈ 78.49 万 × 100 × 61.78 ≈ 48.49 亿元
```

> A 股单位：**手** = 100 股；**万手** = 100 万股。

### 换手率

衡量流通股中有多大比例在当日完成了换手：

```text
换手率 = 成交量 / 流通股本 × 100%
       = 7849000 / 106100000 × 100% ≈ 7.40%
```

| 换手率区间 | 参考含义 |
|:---:|:---|
| < 1% | 交投清淡，缺乏关注 |
| 1%–3% | 正常水平 |
| 3%–7% | 活跃，有资金参与 |
| > 7% | 高度换手，可能是主力出货或抢筹 |

本例换手率 7.40% 配合跌停，属于典型**高位放量跌停**，主力出货信号较强。

### 内盘与外盘

A 股以**主动性**区分内外盘：

```text
外盘（主动买入量） = 33.43 万手  → 买方主动扫单挂卖价成交
内盘（主动卖出量） = 45.07 万手  → 卖方主动砸单挂买价成交

外盘 + 内盘 ≈ 成交量（因含均价成交等，可能有小偏差）
```

- **外盘 > 内盘**：买方力量较强
- **内盘 > 外盘**：卖方力量较强（本例内盘 > 外盘，与跌停吻合）

### 委比

反映当前买卖盘挂单力量对比：

```text
委比 = (委买手数 - 委卖手数) / (委买手数 + 委卖手数) × 100%
委比范围：-100% ~ +100%
```

本例委比 **-100.00%** 表明卖盘几乎清空了买盘（或买单已全被抢光），封跌停后买方挂单为 0，卖方有大量排队待卖。

### 量比

反映当日成交活跃程度，以近期平均水平为基准：

```text
量比 = 当日成交量 / 近 5 日日均成交量
     = 0.90（略低于近 5 日均量）
```

| 量比区间 | 参考含义 |
|:---:|:---|
| < 0.5 | 明显缩量 |
| 0.5–1.5 | 正常量能 |
| 1.5–2.5 | 温和放量 |
| > 2.5 | 显著放量 |

量比 0.90 说明跌停当日量能并未异常放大，封板力度较强。

## 估值类参数

估值参数帮助判断股价是否合理，需结合行业均值横向比较。

### 市盈率（PE，Price-to-Earnings Ratio）

```text
市盈率 = 总市值 / 归母净利润
       = 股价 / 每股收益（EPS）
```

A 股常用三种口径：

| 类型 | 公式 | 中国稀土示例 |
|:---|:---|:---:|
| **静态市盈率**（PE 静） | 总市值 / 上一完整财年净利润 | 366.93 |
| **动态市盈率**（PE 动） | 总市值 / 本年预测净利润（= 最近一季度 EPS × 4） | 114.26 |
| **TTM 市盈率** | 总市值 / 过去 12 个月滚动净利润 | 265.49 |

三种口径差异：

- **PE 静**最保守，基于已公布年报；
- **PE TTM** 最贴近当前盈利能力，业界最常用；
- **PE 动**最激进，依赖利润增速假设。

> 中国稀土 PE TTM = 265.49，说明按当前盈利水平需 265 年才能回本，属于高估值。

### 市净率（PB，Price-to-Book Ratio）

```text
市净率 = 股价 / 每股净资产
       = 59.67 / 4.66 ≈ 12.80
```

- PB < 1：股价跌破净资产，俗称"破净"
- PB 越低越安全（银行/地产常低于 1）
- 轻资产行业（科技/消费）PB 通常较高

### 每股净资产（BPS）

```text
每股净资产 = 股东权益合计 / 总股本
           ≈ 4.66 元/股
```

反映每股对应的账面净资产价值。

### 每股收益（EPS）

```text
每股收益 = 归母净利润 / 总股本
         = 0.13 元/股（当期，即最近一季度）
```

年化 EPS ≈ 0.13 × 4 = 0.52 元/股（动态口径用于计算 PE 动）。

### ROE（净资产收益率）

```text
ROE = 归母净利润 / 股东权益合计 × 100% = 2.84%
```

ROE 是衡量企业资金利用效率的核心指标，巴菲特认为优质企业 ROE 应持续高于 15%。本例 ROE 2.84% 偏低。

### 股息率

```text
股息率 = 每股分红 / 股价 × 100%
```

本例显示 `--`，说明中国稀土近期未分红。

## 股本与市值参数

### 总股本与流通股本

```text
总股本     = 10.61 亿股（公司已发行的全部股份）
流通股本   = 10.61 亿股（可在二级市场自由交易的股份）
```

本例总股本 = 流通股本，说明该公司**没有限售股**，全部股份已上市流通。

若存在大股东限售、高管持股锁定等，则流通股本 < 总股本。

### 总市值与流通市值

```text
总市值   = 股价 × 总股本   = 59.67 × 10.61 亿 ≈ 633.23 亿元
流通市值 = 股价 × 流通股本 = 633.23 亿元（与总市值相同）
```

市值是衡量公司规模的常用指标，A 股分类参考：

| 市值区间 | 分类 |
|:---:|:---|
| < 50 亿 | 小市值 |
| 50–200 亿 | 中小盘 |
| 200–1000 亿 | 中大盘 |
| > 1000 亿 | 大盘蓝筹 |

## 其他标签参数

### ESG 评级

环境（E）、社会（S）、公司治理（G）三个维度的综合评级，由 MSCI 等第三方机构发布：

```text
中国稀土：CCC（MSCI，2026-05-18）
```

MSCI ESG 等级从高到低：AAA → AA → A → BBB → BB → B → CCC，CCC 为最低档。

### 信披考评

上交所 / 深交所对上市公司信息披露质量的年度考评：

```text
评级：A（优秀）
```

分 A（优秀）/ B（良好）/ C（合格）/ D（不合格）四档，A 级说明信披规范、及时。

### 近期涨停 / 1 年涨停次数

```text
最近涨停日：2026-04-29
1 年涨停次数：5 次
```

涨停次数反映股票在过去一年的强势程度，连续涨停往往是主力运作的特征。

## 五档盘口（委托深度）

五档盘口展示当前挂单情况：

```text
卖5  59.71  11.3 万手
卖4  59.70  33.4 万手
卖3  59.69  94.3 万手
卖2  59.68  375  万手
卖1  59.67  1.34 亿手  ← 跌停价，大量封单等待卖出
买1  --      0           ← 无人接盘
```

卖 1 位置挂有 1.34 亿手（即 134 亿股，约 1.26 倍流通股本），而买 1 为 0，这是**封死跌停**的典型特征。

## 综合解读示例

> 中国稀土（000831）于 2026-06-22 盘中创历史新高 66.88 元，次日即以 65.00 元低开、跌停收于 59.67 元，换手率 7.40%，委比 -100%，卖 1 挂单 1.34 亿手。
>
> 这一组合信号通常意味着：**前一日在新高附近主力开始出货，次日散户追涨被套，机构或主力加速离场**。高换手 + 封跌停的模式是 A 股短线见顶后常见的"出货跌停板"形态。
